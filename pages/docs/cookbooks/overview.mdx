# Cookbooks Overview

Cookbooks provide step-by-step integration examples for common use cases and frameworks.

## Available Cookbooks

### LLM Provider Integrations

- **[OpenAI Integration](./openai-integration)**: Integrate Observa with OpenAI's API
- **[Anthropic Integration](./anthropic-integration)**: Track Claude API calls
- **[Vercel AI SDK Integration](./vercel-ai-integration)**: Track Vercel AI SDK (unified SDK for multiple providers)
- **[Function Calling](./function-calling)**: Monitor function calling workflows

### Framework Integrations

- **[Next.js Integration](./nextjs-integration)**: Set up Observa in Next.js applications
- **[Express.js Integration](./express-integration)**: Integrate with Express.js servers

### Use Case Examples

- **[Multi-Turn Conversations](./multi-turn-conversations)**: Track conversational flows
- **[RAG Integration](./rag-integration)**: Monitor Retrieval-Augmented Generation pipelines

## Getting Started

Each cookbook includes:

- **Prerequisites**: What you need before starting
- **Setup Instructions**: Step-by-step configuration
- **Code Examples**: Working code samples
- **Best Practices**: Tips for production use
- **Troubleshooting**: Common issues and solutions

## Prerequisites

Before following any cookbook, ensure you have:

1. **Observa Account**: Sign up at [observa-app.vercel.app](https://observa-app.vercel.app)
2. **API Key**: Get your API key from Settings
3. **SDK Installed**: `npm install observa-sdk`
4. **Basic Knowledge**: Understanding of JavaScript/TypeScript

## Quick Start

If you're new to Observa, start with:

1. **[Quick Start](../quick-start)**: Basic setup
2. **[OpenAI Integration](./openai-integration)**: Most common use case
3. **[Next.js Integration](./nextjs-integration)**: If using Next.js
