# Dashboard Overview

Navigate and use the Observa dashboard effectively.

## Dashboard Features

The Observa dashboard provides comprehensive insights into your LLM application:

- **Overview**: High-level metrics and analytics
- **Traces**: View all tracked LLM calls
- **Sessions**: Manage conversation sessions
- **Issues**: Monitor and resolve problems
- **Costs**: Track spending and usage
- **Settings**: Configure your account

## Navigation

Access the dashboard at [observa-app.vercel.app/dashboard](https://observa-app.vercel.app/dashboard).

### Sidebar Navigation

- **Overview**: Dashboard home with key metrics
- **Traces**: All tracked LLM calls
- **Sessions**: User sessions and conversations
- **Users**: User analytics
- **Issues**: Problems and alerts
- **Costs**: Cost analysis and optimization
- **Settings**: Account configuration

## Overview Page

The overview page shows:

- **Total Traces**: Number of tracked calls
- **Active Sessions**: Current active sessions
- **Cost Summary**: Spending overview
- **Error Rate**: Percentage of failed calls
- **Performance Metrics**: Latency and throughput
- **Time Series Charts**: Trends over time

## Key Metrics

Monitor these key metrics:

1. **Request Volume**: Number of LLM calls
2. **Success Rate**: Percentage of successful calls
3. **Average Latency**: Response time
4. **Cost per Request**: Average spending
5. **Error Rate**: Percentage of errors

## Filtering

Filter data by:

- **Date Range**: Select time period
- **User**: Filter by user ID
- **Conversation**: Filter by conversation ID
- **Model**: Filter by LLM model
- **Status**: Filter by success/error

## Exporting Data

Export data for analysis:

1. Apply filters
2. Click "Export" button
3. Select format (CSV, JSON)
4. Download your data
