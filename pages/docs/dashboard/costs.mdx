# Cost Analysis

Track and optimize your LLM spending.

## Accessing Costs

Navigate to **Costs** in the dashboard sidebar to view spending analytics.

## Cost Overview

The costs page shows:

- **Total Cost**: Overall spending
- **Cost per Request**: Average cost
- **Cost Trends**: Spending over time
- **Model Breakdown**: Costs by model
- **User Breakdown**: Costs by user

## Cost Metrics

Monitor these cost metrics:

1. **Total Spending**: Overall cost
2. **Daily/Weekly/Monthly**: Time-based costs
3. **Cost per Request**: Average spending
4. **Cost per Token**: Token-based pricing
5. **Model Costs**: Breakdown by model

## Cost Breakdown

Analyze costs by:

- **Model**: Which models cost the most
- **User**: Which users generate costs
- **Feature**: Which features are expensive
- **Time**: Cost trends over time
- **Conversation**: Cost per conversation

## Cost Optimization

Optimize costs by:

1. **Model Selection**: Use cheaper models when possible
2. **Token Optimization**: Reduce prompt/response size
3. **Caching**: Cache responses when appropriate
4. **Rate Limiting**: Control usage
5. **Monitoring**: Track and alert on costs

## Budget Alerts

Set up budget alerts:

- Daily spending limits
- Monthly budgets
- Cost threshold alerts
- Anomaly detection

## Exporting Cost Data

Export cost data for analysis:

1. Apply filters
2. Click "Export" button
3. Select format (CSV, JSON)
4. Download your data

## Best Practices

1. **Monitor Regularly**: Check costs frequently
2. **Set Budgets**: Define spending limits
3. **Optimize Usage**: Reduce unnecessary calls
4. **Analyze Trends**: Identify cost patterns
5. **Alert on Anomalies**: Get notified of spikes
