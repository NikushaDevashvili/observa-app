# Get Started with Tracing

Start tracking your LLM calls in minutes. This guide shows you how to set up Observa and track your first trace.

## Prerequisites

- Node.js 16+ installed
- An Observa account ([sign up here](https://observa-app.vercel.app))
- Your LLM provider API key (OpenAI, Anthropic, etc.)

## Step 1: Install the SDK

```bash
npm install observa-sdk
```

## Step 2: Get Your API Key

1. Sign up at [observa-app.vercel.app](https://observa-app.vercel.app)
2. Complete the onboarding
3. Copy your API key from the dashboard

## Step 3: Initialize the SDK

Create a new file (e.g., `index.js`) and initialize Observa:

```javascript
import { init } from 'observa-sdk';

const observa = init({
  apiKey: process.env.OBSERVA_API_KEY,
  apiUrl: 'https://observa-api.vercel.app',
});
```

Set your API key as an environment variable:

```bash
export OBSERVA_API_KEY="your-api-key-here"
```

Or create a `.env` file:

```env
OBSERVA_API_KEY=your-api-key-here
OPENAI_API_KEY=sk-proj-***  # Your LLM provider key
```

## Step 4: Track Your First Call

Wrap your LLM call with `observa.track()`:

```javascript
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

const response = await observa.track(
  {
    query: "What is the capital of France?",
  },
  async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4',
      messages: [
        { role: 'user', content: 'What is the capital of France?' }
      ],
    });
  }
);

console.log(response.choices[0].message.content);
```

Run your code:

```bash
node index.js
```

## Step 5: View Your Trace

1. Open the [Observa Dashboard](https://observa-app.vercel.app/dashboard)
2. Go to **Traces**
3. You should see your tracked call

Click on a trace to see:
- Request and response details
- Performance metrics (latency, tokens)
- Cost information
- Error details (if any)
