# Onboarding Guide

Complete step-by-step guide to get started with Observa.

## Prerequisites

- Node.js 16+ installed
- An Observa account ([sign up here](https://observa-app.vercel.app))
- Your LLM provider API key (OpenAI, Anthropic, etc.)

## Step 1: Sign Up

1. Go to [observa-app.vercel.app](https://observa-app.vercel.app)
2. Click "Sign Up"
3. Enter your email and company name
4. Complete the signup process
5. Copy your API key from the dashboard

## Step 2: Install the SDK

```bash
npm install observa-sdk
```

## Step 3: Initialize SDK

Create a new file (e.g., `observa-setup.js`):

```javascript
import { init } from 'observa-sdk';

export const observa = init({
  apiKey: process.env.OBSERVA_API_KEY,
  apiUrl: 'https://observa-api.vercel.app',
});
```

Set your API key as an environment variable:

```bash
export OBSERVA_API_KEY="your-api-key-here"
```

Or create a `.env` file:

```env
OBSERVA_API_KEY=your-api-key-here
OPENAI_API_KEY=sk-proj-***  # Your LLM provider key
```

## Step 4: Track Your First Event (Auto-Capture)

Use auto-capture with `observeOpenAI()` - the easiest way:

```javascript
import { observa } from './observa-setup';
import OpenAI from 'openai';

// Initialize OpenAI client
const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

// Wrap with Observa - automatic tracking!
const wrappedOpenAI = observa.observeOpenAI(openai, {
  name: 'my-first-app',
  userId: 'user-123',
});

// Use wrapped client - automatically tracked!
const response = await wrappedOpenAI.chat.completions.create({
  model: 'gpt-4',
  messages: [
    { role: 'user', content: 'Hello, Observa!' }
  ],
});

console.log(response.choices[0].message.content);
```

## Step 5: Verify in Dashboard

1. Go to [Dashboard](https://observa-app.vercel.app/dashboard)
2. Navigate to **Traces**
3. You should see your first tracked event
