# Observa Documentation

Observa is an observability platform for LLM applications. Track your LLM calls, monitor performance, analyze costs, and debug issues.

## Get Started

**[Get Started with Tracing](./quick-start)** - Track your first LLM call in minutes

## What is Observa?

Observa provides observability for LLM applications. It automatically tracks your LLM calls, captures performance metrics, monitors costs, and helps you debug issues.

### Key Features

- **Automatic Tracking**: Wrap your LLM calls with `observa.track()` to capture everything
- **Real-time Dashboard**: View traces, sessions, and analytics
- **Cost Monitoring**: Track spending and optimize usage
- **Error Detection**: Get alerted to problems automatically
- **User Analytics**: Understand how users interact with your LLM features

## How It Works

1. **Install the SDK**: `npm install observa-sdk`
2. **Initialize**: Set up the SDK with your API key
3. **Track Calls**: Wrap your LLM calls with `observa.track()`
4. **View Data**: See traces in the dashboard
