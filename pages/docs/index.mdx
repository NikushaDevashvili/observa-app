# Observa Documentation

Welcome to Observa - the complete observability platform for LLM applications.

## What is Observa?

Observa is an observability and monitoring platform designed specifically for LLM (Large Language Model) applications. Track your LLM calls, monitor performance, analyze costs, and debug issues all in one place.

## Architecture

Observa consists of three main components:

1. **observa-sdk**: npm package for tracking LLM calls in your application
2. **observa-api**: Backend API service that processes and stores your data
3. **observa-app**: Web dashboard for visualizing and analyzing your data
